## Getting started
- Begin by downloading the git using the command "git clone https://github.com/BlueBirdMC/Server.git" or by selecting the zip file from the code dropdown.Â 
- Next, open a terminal inside the download or git folder and type "npm install".
- The last step is to run "node ." or "node index.js".
